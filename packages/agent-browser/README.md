# Agent Browser

Headless browser automation (agent-browser v0.27.0)

## Tools
- `browser_navigate` - Navigate to URL
- `browser_click` - Click element
- `browser_type` - Type text
- `browser_extract` - Extract text
- `browser_screenshot` - Take screenshot
- `browser_session_start/end` - Manage sessions
- `browser_doctor` - Verify installation
