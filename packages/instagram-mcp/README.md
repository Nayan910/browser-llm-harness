# Instagram MCP Server

Instagram Graph API v19.0 MCP server for the Browser LLM Harness.

## Tools
- `instagram_get_media` - Get post/reel details
- `instagram_list_media` - List recent posts
- `instagram_get_comments` - Get comments
- `instagram_get_metrics` - Get engagement metrics
- `instagram_search_hashtag` - Search by hashtag
- `instagram_get_profile` - Get business profile
- `instagram_analyze_post_url` - Analyze from URL

## Setup
1. Copy `config/default.env` to `.env`
2. Fill in your Instagram credentials
3. Run: `npm start`
